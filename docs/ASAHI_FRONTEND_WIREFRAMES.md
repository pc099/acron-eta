## 8. COMPONENT STATES

### Button Variants
- **Primary**: Orange background, white text, dark orange on hover
- **Secondary**: Light gray background, dark text
- **Outline**: Orange border, orange text, light orange on hover
- **Ghost**: Text only, light orange on hover
- **Disabled**: 50% opacity, no interaction

### Form Components
- **Text Input**: Subtle gray border, orange border on focus
- **Textarea**: Same as input, with character counter
- **Toggle**: Orange when ON, gray when OFF
- **Slider**: Orange track and thumb
- **Dropdown**: Orange highlight on selection

### Loading States
- Skeleton loaders with gray placeholders
- Spinning loader icon
- Progress bars showing percentage

### Error/Success States
- **Error**: Red/pink background, red icon
- **Success**: Green background, green checkmark
- **Warning**: Yellow/amber background
- **Info**: Blue background
